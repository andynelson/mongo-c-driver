mongoc_database_add_user(3)
===========================


NAME
----
mongoc_database_add_user, mongoc_database_remove_user, mongoc_database_remove_all_users - manage users in a database


SYNOPSIS
--------
[source,c]
-----------------------
bool
mongoc_database_add_user (mongoc_database_t *database,
                          const char        *username,
                          const char        *password,
                          const bson_t      *roles,
                          const bson_t      *custom_data,
                          bson_error_t      *error);

bool
mongoc_database_remove_user (mongoc_database_t *database,
                             const char        *username,
                             bson_error_t      *error);

bool
mongoc_database_remove_all_users (mongoc_database_t *database,
                                  bson_error_t      *error);
-----------------------


DESCRIPTION
-----------
The _mongoc_database_add_user()_ function shall create a new user with access
to 'database'.

username:: a non-optional string holding the username

password:: an optional string holding the password

roles:: an optional bson_t of roles

custom_data:: an optional bson_t of custom data to store

error:: an optional error pointer to store any error data

The _mongoc_database_remove_user()_ function shall remove the specified user
from 'database'.

The _mongoc_database_remove_all_users()_ function shall remove all users
from 'database'.

RETURN VALUE
------------
The _mongoc_database_add_user()_ function returns true if the user was
successfully added.  In the event of an error it returns false an 'error' is
set.

The _mongoc_database_remove_user()_ and _mongoc_database_remove_all_users()_
functions shall return true if successful, otherwise false and _error_ is set.

ERRORS
------
Errors are returned through 'error' and may vary based on the server version.
